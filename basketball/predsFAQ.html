Projection Model FAQ

This is an FAQ for my draft projection model app.

Use the "dataset" pulldown to toggle between "Draft", "NCAA", and "Euro." The draft dataset includes retrodictions for all players drafted since 1990, as well as predictions for the 2015 draft class which I will update as the NCAA season progresses. The NCAA and Euro datasets include additional players and list each season separately.

Use the "Seasons" slider to select the range of years you want to include in your browsing, then further filter the list by increasing the minutes threshold. If you want to sort columns by highest or lowest score, simply click on the column header of the stat you are interested in. If you are just looking for a specific player, I recommend the search function on the right side of the page.

EWP:

"Expected Wins Peak" is a quasipoisson regression model used to project how well a player will perform at the peak of his NBA career. NCAA and international prospects are projected using two similar but distinct models. The NCAA model uses age, experience, MPG, SOS, Height, Weight, and basic per-possession box-score statistics fit from all players to play in both the NCAA and NBA since 1990. I have outlined the international model in detail in an article series ( here, here, and here).

The values in the "EWP" column give the number of "wins" each player is projected to be worth at his peak. Wins are defined by a combined two-year average of Win Shares and RAPM-wins.

HUM:

The "Humble" model has the same ultimate goal as the EWP model, project a player's win peak. However, it differs in two significant ways. 1) The Humble model uses a randomForest machine-learning process to fit the value of each variable instead of classic linear regression. 2) The Humble model includes a transformed variable that captures concensus scouting opinion of each player using actual draft order and popular draft big-boards. The Humble model is currently only calculated for NCAA prospects.

The values in the "HUM" column should be interperted in the same way as the EWP column.

BUST, BNCH, STRT, STUD, STAR:

These columns list the outputs of a multinomial regression used to project the likelihood that a player will ultimately reach each tier of NBA performance:

`Bust': Never worth more than 2.5 wins in a season.

`Bench warmer': Peak value between 2.5 and 5 wins.

`Starter': Peak value between 5 and 7.5 wins.

`Stud': Peak value between 7.5 and 10 wins.

`Star': Ultimately worth more than 10 wins in a season.

This model uses a player's combined CMB score from the previous metrics, but also includes no-step veritcal, and wingspan / height as additional variables.
